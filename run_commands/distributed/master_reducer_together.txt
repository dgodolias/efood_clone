# Σενάριο: Master + Reducer σε ένα μηχάνημα, Worker σε άλλο μηχάνημα
# Αυτό το αρχείο περιέχει οδηγίες για να τρέξετε το σύστημα με τον Master και τον Reducer
# στο ίδιο μηχάνημα και τον Worker (ή Workers) σε άλλο(α) μηχάνημα(τα).

# ===== ΟΔΗΓΙΕΣ =====

# 1. Πρώτα εκκινήστε τον Worker στο δεύτερο μηχάνημα:
#    Στο μηχάνημα που θα λειτουργεί ως Worker, εκτελέστε την παρακάτω εντολή:
#    Αντικαταστήστε το PORT με τον αριθμό θύρας (π.χ. 8081)

java -cp ../build/classes/java/main com.example.backend.Worker PORT

# 2. Στη συνέχεια εκκινήστε τον Reducer στο πρώτο μηχάνημα:
#    Στο μηχάνημα που θα λειτουργούν ο Master και ο Reducer, εκτελέστε την παρακάτω εντολή:
#    Αντικαταστήστε το WORKER_IP με την IP διεύθυνση του μηχανήματος Worker
#    και το WORKER_PORT με τη θύρα του Worker (π.χ. 8081)

java -cp ../build/classes/java/main com.example.backend.Reducer WORKER_IP:WORKER_PORT

# 3. Τέλος, εκκινήστε τον Master στο ίδιο μηχάνημα με τον Reducer:
#    Αντικαταστήστε το WORKER_IP με την IP διεύθυνση του μηχανήματος Worker,
#    το WORKER_PORT με τη θύρα του Worker (π.χ. 8081)
#    και το REDUCER_IP με την IP διεύθυνση του μηχανήματος που τρέχει τον Reducer (localhost ή 127.0.0.1 αν είναι στο ίδιο μηχάνημα)

java -cp ../build/classes/java/main com.example.backend.Master --distributed --reducer localhost --workers WORKER_IP:WORKER_PORT

# ===== ΠΑΡΑΔΕΙΓΜΑ =====

# Έστω ότι έχουμε:
# - Μηχάνημα A (IP: 192.168.1.100): Θα τρέχει Master + Reducer
# - Μηχάνημα B (IP: 192.168.1.101): Θα τρέχει Worker

# Στο μηχάνημα B, εκτελούμε:
# java -cp ../build/classes/java/main com.example.backend.Worker 8081

# Στο μηχάνημα A, εκτελούμε πρώτα:
# java -cp ../build/classes/java/main com.example.backend.Reducer 192.168.1.101:8081

# Και στη συνέχεια (σε άλλο τερματικό) στο μηχάνημα A, εκτελούμε:
# java -cp ../build/classes/java/main com.example.backend.Master --distributed --reducer localhost --workers 192.168.1.101:8081
